## 📌 项目简介

CryptoTrader 是一款**赛博朋克风格**的加密货币合约交易日志系统，帮助交易者记录、统计和分析合约交易数据。

### 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 💰 资金管理 | 输入总资金，实时计算权益和收益率 |
| 📝 交易记录 | 添加/编辑/删除交易记录，支持多种交易对 |
| 📊 数据统计 | 总交易次数、挂单/吃单统计、手续费汇总 |
| 📈 可视化图表 | 权益曲线图、交易类型分布饼图 |
| 🎯 盈亏分析 | 最大盈利、最大亏损、胜率计算 |
| 🔄 数据持久化 | 支持后端 API 存储，数据不丢失 |

---

## 📖 使用指南

### 添加交易记录

1. 选择交易对（如 BTC/USDT）
2. 输入买入价格（开仓价）
3. 输入仓位数量
4. （可选）输入平仓价格，计算盈亏
5. 选择交易类型：挂单 / 吃单
6. 选择方向：做多 / 做空
7. 点击「添加记录」

### 查看统计

- **总交易次数**: 所有交易记录数量
- **挂单/吃单次数**: 按交易类型分类统计
- **总手续费**: 吃单手续费 0.04%
- **当前权益**: 总资金 - 手续费 + 盈亏
- **收益率**: (当前权益 - 总资金) / 总资金 × 100%
- **最大盈利**: 已平仓交易中盈利最多的一笔
- **最大亏损**: 已平仓交易中亏损最多的一笔
- **胜率**: 盈利交易数 / 已平仓交易数 × 100%

## 🔧 API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/data` | 获取所有数据 |
| POST | `/api/capital` | 设置资金 |
| GET | `/api/trades` | 获取交易列表 |
| POST | `/api/trades` | 添加交易 |
| PUT | `/api/trades/:id` | 更新交易 |
| DELETE | `/api/trades/:id` | 删除交易 |
| GET | `/api/stats` | 获取统计信息 |

---

## 📝 注意事项

- 挂单手续费：**0%**
- 吃单手续费：**0.04%**
- 盈亏计算：
  - 做多：(平仓价 - 买入价) × 仓位
  - 做空：(买入价 - 平仓价) × 仓位
